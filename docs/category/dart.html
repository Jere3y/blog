spread operator (...)

不知道咋翻译这个操作符。

Dart 2.3 新增 ... 操作符。

使用 ... 可以简单方便的把一个列表的所有元素插入到另一个列表中。

var list = [1, 2, 3];
var list2 = [0, ...list];
assert(list2.length == 4);

上面的代码 ...list，将 list 分解开，插入到了 list2 中，执行完代码后。

list2 的值为 [0,1,2,3]。

标准的 Base64 编码

先给出 RFC3548 给出的 Base64 定义。

The Base 64 encoding is designed to represent arbitrary sequences of octets in a form that requires case sensitivity but need not be humanly readable.

可以看出，Base64 编码算法主要有几个特点：

  • 表示任意 8 位 bit
  • 不需要人类可读
  • 编码和解码简单
  • 编码后数据变大约 33%
